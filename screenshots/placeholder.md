# Screenshots

Add these screenshots to this folder:

- `dashboard.png` — overview tab showing charts and stats
- `live-feed.png` — live feed tab with hits coming in
- `credentials.png` — credentials tab showing captured passwords
- `system-events.png` — system events tab

To take them: open the dashboard at http://localhost:7777, let it run for a bit to collect some data, then screenshot each tab.
